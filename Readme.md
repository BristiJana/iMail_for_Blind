# iMail - Email for Blind

A Django application that enables you to access GMail through voice.

### Installation :- ###

      1. Download the project.
      2. Unzip the file.
      3. Open folder in terminal
      4. Install necessary reuirements using the command "pip3 install -r requirements.txt"
      5. Run the virtual environment using the command ". venv/bin/activate"
      6. Open folder "bmail" using command "cd bmail"
      7. Replace the credentials file with your google cloud credentials.
      8. Run the python server using command "python manage.py runserver"
